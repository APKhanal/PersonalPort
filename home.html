Projects

MixNMath

MixNMath is a web application that tests your quantitative skills with numbers. The goal is to find a way to arrange numbers and symbols and get the goal number. Every day, a new seeded number is given as a challenge as well that the users can collectively solve. I am currently making a mobile version to expand MixNMath's potential

Appstractify

Appstractify is an application which attempts to demystify data analysis by visualizing data simply and allowing users to synthesize data for cross analysis. This, accompanied by new features that let users talk to their data using the Chatbot Edison, makes the complex a lot simpler.
